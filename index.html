Add it in Gmail
  1. Open Gmail → Settings (gear) → See all settings.
  2. Under General, find Signature.
  3. Create or edit a signature, click in the editor, then paste (⌘+V / Ctrl+V).
  4. If formatting looks wrong, use Copy HTML for Gmail above, or paste the raw HTML via a browser extension that inserts HTML — plain paste usually works in the rich text box.
